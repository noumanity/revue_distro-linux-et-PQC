# comprendre les distributions linux

## Contexte

Le coeur de la présentation, c'est les distributions linux: comment elles intègrent l'enjeux du PQC.

Il est donc important de comprendre ce domaine, les distributions et leurs spécialités et la manière dont ils leur développement fonctionne, comment ils gèrent leur roadmap, etc.

Mais... le sujet est très vaste. Donc, nous avons besoin de collecter et de syntétiser beaucoup d'information afin quelle soit digestible pour un humain.
